Apply for Credentials

myNAATI steps you through the process of completing an application for credentials.  At the conclusion of the process you can print the largely completed forms for submission to NAATI along with supporting documentation.

You can apply online if you want to:

•     Take a test set by NAATI; or

•     Apply based on your completion of an approved Australian course.

Contact NAATI to request all other application types.

 

To complete the application forms online:

1.  If you have a Customer Number, log on to expedite the application process.  Note:  If you have not yet registered with myNAATI click on the Become One Now link to register.

2.  Select Applications and review the information provided on the screen

3.  Select By Testing or By Australian Course

4.  Review the eligibility, application and cost information provided

5.  Click Apply Now

6.  Complete the information requested on each page

 

7.  Use the Cancel, Previous and Next buttons to move through the wizard or click on a completed step to review your entries

8.  Click on the Add Request or Add Test buttons to request as many credentials as required

9.  To save changes made to a previously completed screen, e.g. Residency Status, click the Next button

10. Click Download Form to view the application form in a separate window

A unique reference number is allocated to the application

11. Click Print to print your copy of the application

12. Click Done to return to the main menu

13. Follow the instructions on the printed application form to complete your application for credentials
